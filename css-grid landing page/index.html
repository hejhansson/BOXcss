box-sizing:border-box; & display:inline-block;

Because of the box-sizing:border-box; you can add padding to your elements without destroying the layout. By using display:inline-block; you don't have to care about clearing floats you can use floats as they were originally intended for.

So, what's included?

The grid.
Typography.
Separate css with various CSS3-buttons.
A simple nomalize.css.

All above is in it's very own CSS-file so you can choose to use whichever you like

How do I use it?

Include the files you want to use. In the grid, the "box-12" has 100% width of the container div and the box-6 have 50%. If you can handle basic math you'll figure out the rest by yourself.

<div class="container">
<div class="box-12">
</div>
<div class="box-6">
</div>
<div class="box-6">
</div>
</div>
